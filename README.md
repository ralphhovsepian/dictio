# dictio
Search for dictionary words and bookmark them. Made with React js, Materialize css, Merriam webster api, and Firebase

## How to install

```
git clone https://github.com/ralphhovsepian/dictio.git
```

## How to run

```
npm install
npm start
```

Make sure to add your own api keys
